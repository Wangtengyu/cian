# 此岸 CIAN

> 此岸已渡，彼岸未知

🎮 **像素风人生选择产品** — 每天一道社会热点选择题，选择后看到一条路的结局，另一条路需要看广告解锁。

## 产品特点

- 🎮 **像素风设计**：复古 8-bit 游戏视觉风格
- 📅 **每日更新**：每天一道新的社会热点选择题
- ⏰ **时间线结局**：从第 1 年到第 10 年的完整人生轨迹
- 🔓 **广告解锁**：通过观看广告解锁「彼岸」的结局
- 📊 **真实共鸣**：不做简单的好坏判断，展现人生的复杂性

## 技术栈

- **前端**: Next.js 14+ (App Router)
- **样式**: Tailwind CSS
- **字体**: Press Start 2P (像素字体)
- **部署**: Vercel

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```


## 功能概览

### 首页
- 像素风 Logo 与 Slogan
- 今日社会热点选择题
- 参与人数实时显示

### 选择页
- 两个选项（A/B）
- 你选择的路的时间线结局（第1年、第3年、第5年、第10年）

### 广告解锁
- 模拟广告观看（可替换为真实广告 SDK）
- 解锁「彼岸」结局

## 题目预览

产品文档中包含 10 道紧跟社会热点的选择题：

1. 大厂裁员 - 主动被裁 vs 继续苟着
2. 房价困境 - 趁降价买房 vs 继续观望
3. AI 焦虑 - 转型学 AI vs 继续老本行
4. 35 岁危机 - 主动离职 vs 赖着不走
5. 城市选择 - 北上广漂着 vs 回老家
6. 婚恋困境 - 条件合适 vs 选择初恋
7. 生育选择 - 生娃 vs 丁克
8. 考公抉择 - 互联网 vs 公务员
9. 跳槽选择 - 跳槽搏一把 vs 继续苟着
10. 躺平还是卷 - 继续卷 vs 躺平

## 部署

项目已配置 Vercel，可直接部署：

1. Fork 本仓库
2. 在 [Vercel](https://vercel.com) 导入项目
3. 点击 Deploy

或者使用 GitHub Actions 自动部署。
