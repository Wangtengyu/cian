# 此岸 CIAN

> 人生是一场选择，你只能走一条路。

像素风人生选择产品，每天一道社会热点选择题，选择后看到一条路的结局，另一条路需要看广告解锁。

## 产品特点

- 🎮 **像素风设计**：复古8-bit游戏视觉风格
- 📅 **每日更新**：每天一道新的社会热点选择题
- ⏰ **时间线结局**：从第1年到第10年的完整人生轨迹
- 🔓 **广告解锁**：通过观看广告解锁"彼岸"的结局
- 📊 **真实共鸣**：不做简单的好坏判断，展现人生的复杂性

## 技术栈

- **前端**: Next.js 14+ (App Router)
- **样式**: Tailwind CSS
- **字体**: Press Start 2P (像素字体)
- **部署**: Vercel

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
